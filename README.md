# 策略组自动整理

GUI.for.SingBox 插件，用于在核心启动前按节点名称自动生成国家策略组，并插入未隐藏策略组。

## 功能

- 生成 `🇭🇰 HK Group`、`🇹🇼 TW Group`、`🇯🇵 JP Group`、`🇺🇸 US Group`、`🇦🇺 AU Group`、`🇩🇪 DE Group`。
- 台湾组默认只包含名称中有 `CN2` 或 `CFT` 的节点。
- 自动跳过 GFS 中隐藏的策略组。
- 自动清理旧国家组残留引用，避免 `dependency not found`。
